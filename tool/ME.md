## 내가 사용하는 Tool

1. [Arc](https://arc.net/) - Chromium 기반 웹 브라우저로, Spaces와 Profiles를 통해 온라인 활동을 깔끔하게 정리할 수 있다.
2. [Conductor](https://www.conductor.build/) - 여러 AI 코딩 에이전트(Claude Code, Codex 등)를 병렬로 실행하고 코드 변경 사항을 리뷰/머지할 수 있는 Mac 전용 앱이다.
3. [Alt](https://www.altalt.io/ko/) - 강의와 회의를 자동으로 필기하고 요약해주는 온디바이스 AI 노트 앱이다.
4. [OrbStack](https://orbstack.dev/) - Docker Desktop을 대체하는 경량 고속 앱으로, macOS에서 Docker 컨테이너와 Linux 가상 머신을 빠르고 가볍게 실행할 수 있다.
5. [Raycast](https://www.raycast.com/) - macOS의 Spotlight을 대체하는 생산성 런처로, 앱 실행, 클립보드 관리, 스니펫, AI 등 다양한 확장 기능을 제공한다.
6. [Warp](https://www.warp.dev/) - Rust로 작성된 차세대 터미널 에뮬레이터로, AI 기반 명령어 추천과 IDE 스타일 편집 기능을 지원한다.
7. [Raindrop.io](https://raindrop.io/) - 웹에서 발견한 콘텐츠를 저장, 정리, 재발견할 수 있는 올인원 북마크 매니저이다.
8. [Excalidraw](https://excalidraw.com/) - 손으로 그린 듯한 스타일의 다이어그램과 스케치를 만들 수 있는 오픈소스 화이트보드 도구이다.
9. [lazygit](https://github.com/jesseduffield/lazygit) - Git 명령어를 직관적인 터미널 UI로 쉽게 사용할 수 있게 해주는 도구이다.
10. [Trancy](https://www.trancy.org/) - YouTube, Netflix 등 영상에 이중 자막을 제공하고, AI 기반 발음 평가·문법 분석·단어장·딕테이션 등 다양한 언어 학습 기능을 지원하는 브라우저 확장 프로그램이다.
